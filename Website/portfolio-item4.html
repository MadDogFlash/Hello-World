Slack Bot

A bot

The scope of this project was to work with slack's api, node.js, express, and google's sheets api to create a working slackbot. The goal of the project was for the slack bot to be able to read and write to a specified google sheet via slack commands. We needed to use ngrock to host a server for this as slacks api requires https in order to utilize slack commands. We also ended up installing several dependecies including a edit-google sheets api dependency for node.js to simplify that slackbots interaction with google sheets. If we were to continue with project we would likely remove this depedency and rewrite code in favor of using just the google sheets api, as it offers more flexiablity (but is far less readable).

My hand in this group project involved in creating the wireframe for stylistics choice.
